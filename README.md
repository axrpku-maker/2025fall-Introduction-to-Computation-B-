关于如何学：  

我没有统计过，不过我感觉大多数时候我每周课下投入的时间可能是不到8小时的（机考前一个月开始可能才达到吧……），所以水平跟大佬们还是有很大的差距的。以下只说我个人情况，希望给跟我情况差不多（不擅长，不热爱，但是想尽可能学好）的同学一点点参考吧。  

上课和作业就不说了，这里只说一下刷题。前期（大概到矩阵和贪心）我主要是在做book my flight上相应的题（语法类、数学类、贪心），T题基本不做（做不出来……）。中期（从递归到机考前一个月）主要是在跟每日选做，做了七七八八。我比较喜欢拿一整块时间把一周的都做了。老师选题一般是讲到哪就放几道知识点相同的，个人感觉这样集中练习还挺有助于掌握知识点的。这个时候学会了看题解，有些（很多）时候做不出来就去看题解然后学。最后复习期间除了各个知识点都做了一点题，还通过刷leetcode上的题集中补了几个前期学习中的漏洞，比如回溯，居然还真考了哈哈哈。
